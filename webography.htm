Brief Explanations for the Web Site Resources Listed

www.peoi.org
This website is a great resource for tags, formatting images, and forms.

www.w3schools.com
This page is a great resource for tutorials and references. They also provide you with a space to practice and implement HTML.

https://developer.mozilla.org
This website has a ton of resources for understanding elements of HTML. This site also has a search bar where you can type a bit of code into and it breaks down the meaning of the code, how to use it, Ect. There is an interactive area to test the code and see what happens in real time.

www.geeksforgeeks.org
This website is a great resource for not only HTML5 but also other languages like Java, Python, JavaScript, C++ and more! There are many tutorials and courses as well.

https://www.learn-html.org
This website gives you step by step tutorials of HTML as well as more advanced tutorials. They also have beginner and advanced tutorials of CSS.

https://html.com
This website offers you what features of HTML are deprecating and what to use in replacement of those elements. There are separate pages of new additions of HTML5 as well as what is no longer supported in HTML5.

https://www.tutorialspoint.com
his website is a great resource for elements of HTML. There are multiple examples on each topic and a place to practice and see in real time how the code works. There is also a page dedicated to depreciating features and replacements for them.

https://blog.hubspot.com
his is a great resource for understanding how to change color using CSS.

https:html.spec.whatwg.org
This page gives you a few examples of other cool things HTML can do such as autocomplete for typing into a space.

https://web.dev
This website runs through different elements of HTML. There are interactive sections that you can see how the code affects the outcome of the formatting. Some of the topics this site covers are metadata, document structure, semantic HTML, links, navigation, audio and video, and so much more.
